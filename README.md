# 🟢 Golden Yield Mart Management System

Production-ready shop management system built for real-world retail operations.  
Currently in active use for managing inventory, sales, and customer debt in a live business environment.

---

## 📊 Overview

Golden Yield Mart Management System is a full-stack business application designed to handle daily retail operations efficiently.

It supports real-time inventory tracking, sales processing, debt management, and business analytics, helping store owners manage and grow their operations with accurate data.

---

## 🚀 Key Features

### 🧾 Inventory Management
- Real-time stock tracking
- Multi-unit product support (base unit + sale units)
- Automatic stock deduction based on sales

### 💰 Sales System (POS)
- Fast sales processing
- Dynamic pricing rules (e.g., bulk pricing)
- Automatic calculation of totals and profit

### 📊 Debt Management
- Customer debt tracking
- Payment recording and adjustments
- Debt ledger linked to sales

### 📈 Reports & Analytics
- Daily, weekly, and monthly sales insights
- Revenue, cost, and profit tracking
- Business performance dashboard

---

## 🧠 Engineering Highlights

- Designed with scalable backend architecture
- Handles real-world edge cases (stock conversion, pricing rules, debt tracking)
- Database-driven system (no reliance on local state)
- Optimized for performance and data consistency

---

## 🛠 Tech Stack

**Frontend:**
- Next.js (App Router)
- TypeScript
- Tailwind CSS

**Backend:**
- Node.js (API routes)
- Prisma ORM
- PostgreSQL (Neon)

**Other Tools:**
- Zustand (state management)
- Vercel (deployment)

---

## ⚙️ Architecture

- Full-stack Next.js application
- API-driven backend structure
- Prisma schema for data modeling
- Modular and scalable code organization

---

## 🌍 Live Usage

This system is actively used in a physical retail shop for:
- Daily sales transactions
- Inventory management
- Customer debt tracking
- Business reporting
